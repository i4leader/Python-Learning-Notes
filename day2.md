### cat合并文件、grep查找文件内容
![cat操作](images/day2-1.jpg)
```
cat test.txt 
```
将test.txt中的内容显示出来
```
    cat test.txt test02.txt
```
先显示text.txt 中的内容再显示text02.txt中的内容
```
    cat test.txt test02.txt > text03.txt
```
将test.txt 和test02.txt中的内容合并到text03.txt 中。
